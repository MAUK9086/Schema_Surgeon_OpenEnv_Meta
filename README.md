## 1. Idea Description

### The Real-World Problem

In production NoSQL ecosystems (MongoDB, DynamoDB, Firestore), **schema drift** is a silent killer of data quality. Because NoSQL allows flexible, schemaless documents, different application versions write different document shapes over time:

- `user_id` vs `uid` vs `u_id` — three keys for the same concept
- `age: 25` vs `age: "25"` — integer stored as string after a serializer bug
- `version: "v1"` vs `metadata: { version: "v1" }` — nesting added/removed arbitrarily

This drift creates downstream failures in analytics pipelines, ML feature stores, and API contracts. Current tools require manual scripting by engineers for every migration.

### Our Solution

**Schema Surgeon** is a deterministic, reproducible OpenEnv RL environment where an AI agent acts as an autonomous **Database Reliability Engineer (DRE)**. The agent:

1. Observes a "messy" document collection (50 documents with deliberate drift)
2. Applies **global, idempotent transformations** (rename, cast, flatten, delete)
3. Is scored by how many documents pass a strict **JSON Schema (Draft 7)** target

The environment is designed so that:
- A perfect agent can always reach `S = 1.0` (no deadlocks)
- Reward is incremental (not sparse), giving meaningful training signal
- All randomness is eliminated via pre-committed static JSON files

## 4. Dataset Design & Generation

### Philosophy: Static, Pre-Committed Files

**CRITICAL DECISION:** The datasets `task1.json`, `task2.json`, `task3.json` are **pre-generated once and committed to the repository**. `generate_data.py` is a developer utility only and is NEVER called at runtime. This guarantees:
- Identical data for all judges, teammates, and CI runs
- No Python version–dependent `random` output differences
- Reproducible baseline scores

### Document Volume

- **50 documents per task**
- **First 10 documents ("Golden 10"):** Manually curated. Each covers at least one of: key naming drift, type drift, nesting drift. The agent always sees these 10 in observations.
- **Documents 11–50:** Generated by `generate_data.py` with `random.seed(42)`, then frozen.

### Schema of a "Perfect" Base Document

```json
{
  "user_id": "user_001",
  "age": 25,
  "price": 19.99,
  "version": "v1"
}
```

### Mutation Rules Applied During Generation

| Mutation | Rule | Probability |
|---|---|---|
| Key rename | `user_id` → `uid` or `u_id` | 40% of docs |
| Type drift (age) | `age: 25` → `age: "25"` | 30% of docs |
| Type drift (price) | `price: 19.99` → `price: "19.99"` | 30% of docs |
| Nesting drift | `version: "v1"` → `metadata: { version: "v1" }` | 50% of docs |

]


def generate_bulk(golden_10: list, task_id: int) -> list:
    """
    Generate 50 documents: first 10 from golden_10, remaining 40 procedurally.
    Uses random.seed(SEED) for reproducibility of the bulk portion.

    Args:
        golden_10: Manually curated list of 10 documents.
        task_id: Which task (1, 2, or 3) to generate for.

    Returns:
        List of 50 documents.
    """
    random.seed(SEED)
    docs = copy.deepcopy(golden_10)

    for i in range(10, NUM_DOCS):
        uid_key = random.choice(["uid", "u_id", "user_id"])
        age_val = random.randint(18, 65)
        price_val = round(random.uniform(1.0, 200.0), 2)

        if task_id == 1:
            doc = {
                uid_key: f"user_{i:03d}",
                "age": age_val,
                "price": price_val,
                "version": random.choice(["v1", "v2"]),
            }
        elif task_id == 2:
            doc = {
                "user_id": f"user_{i:03d}",
                "age": str(age_val) if random.random() < 0.3 else age_val,
                "price": str(price_val) if random.random() < 0.3 else price_val,
                "version": random.choice(["v1", "v2"]),
            }
        else:
            version_val = random.choice(["v1", "v2"])
            doc = {
                uid_key: f"user_{i:03d}",
                "age": str(age_val) if random.random() < 0.3 else age_val,
                "price": str(price_val) if random.random() < 0.3 else price_val,
            }
            if random.random() < 0.5:
                doc["metadata"] = {"version": version_val}
            else:
                doc["version"] = version_val

        docs.append(doc)

    return docs


def main():
    """Entry point. Generates and writes all three task datasets."""
    os.makedirs(OUTPUT_DIR, exist_ok=True)

    datasets = {
        "task1": generate_bulk(GOLDEN_10_TASK1, 1),
        "task2": generate_bulk(GOLDEN_10_TASK2, 2),
        "task3": generate_bulk(GOLDEN_10_TASK3, 3),
    }

    for name, data in datasets.items():
        path = os.path.join(OUTPUT_DIR, f"{name}.json")
        with open(path, "w") as f:
            json.dump(data, f, indent=2)
        print(f"Written: {path} ({len(data)} docs)")


if __name__ == "__main__":
    main()

## 5. Environment Design

### `models.py` — Pydantic v2 Typed Models

```python
"""
models.py
---------
Pydantic v2 typed models for OpenEnv API compliance.
Defines Action, Observation, and Reward schemas used by both
the server (SchemaSurgeon_environment.py) and client (client.py).
All fields must be JSON-serializable.
"""

from typing import Any, Dict, List, Optional
from pydantic import BaseModel, Field


class SchemaAction(BaseModel):
    """
    Represents one agent action dispatched to the environment.

    Fields:
        action_type: One of rename_and_merge | cast_type | flatten_field
                     | delete_key | terminate
        params: Dict of keyword arguments for the chosen action.

    Examples:
        {"action_type": "rename_and_merge", "params": {"source": "uid", "target": "user_id"}}
        {"action_type": "cast_type", "params": {"key": "age", "target_type": "int", "default_value": 0}}
        {"action_type": "flatten_field", "params": {"parent_key": "metadata", "child_key": "version"}}
        {"action_type": "delete_key", "params": {"key": "legacy_field"}}
        {"action_type": "terminate", "params": {}}
    """
    action_type: str = Field(..., description="Action name to dispatch")
    params: Dict[str, Any] = Field(default_factory=dict, description="Action parameters")


class SchemaObservation(BaseModel):
    """
    What the agent sees at each step.

    Fields:
        sample_docs: First 10 documents (Golden Sample). Always same order.
        global_keys: Dict mapping every key to its presence ratio (0.0-1.0).
        target_schema: The JSON Schema the agent must satisfy.
        step_count: Current step number (1-indexed after first step).
        max_steps: Hard cap for this task.
        current_score: Float 0.0-1.0, fraction of docs passing the schema.
        last_action_status: "success" | "no_op" | "error: <message>" | "reset"
    """
    sample_docs: List[Dict[str, Any]]
    global_keys: Dict[str, float]
    target_schema: Dict[str, Any]
    step_count: int
    max_steps: int
    current_score: float
    last_action_status: str


class SchemaReward(BaseModel):
    """
    Reward model returned with each step result.

    Fields:
        value: Delta reward = max(0, (S_t - S_{t-1})) * 10
        score: Current global score S_t (0.0-1.0)
    """
    value: float
    score: float
```

### `server/SchemaSurgeon_environment.py` — Core Environment

```python
"""
SchemaSurgeon_environment.py
-----------------------------
Core OpenEnv environment class for Schema Surgeon.
Extends openenv.core.Environment and implements reset(), step(), state().

All state is held in-memory as a Python list of dicts (simulated MongoDB collection).
No database. No randomness. All data loaded from static JSON files.

Key design decisions:
- copy.deepcopy() on collection before each mutating action for safe rollback.
- Stagnation counter auto-terminates after STAGNATION_LIMIT non-improving steps.
- Protected keys (from schema 'required') block delete_key actions.
- Score recomputed from scratch after every step via grader.calculate_score().
"""

import copy
import json
from typing import Any, Dict, List, Tuple

from openenv.core import Environment  # adjust import path if needed
from server.actions import dispatch_action
from server.grader import calculate_score
from server.models import SchemaAction, SchemaObservation, SchemaReward
from server.obs import build_observation
from server.tasks import get_task

STAGNATION_LIMIT: int = 5
GOLDEN_SAMPLE_SIZE: int = 10


class SchemaSurgeonEnvironment(Environment):
    """
    OpenEnv environment for autonomous NoSQL schema migration.

    The agent must apply a sequence of global transformations to migrate
    a collection of 50 documents to satisfy a target JSON Schema.

    Attributes:
        task_id: Active task identifier ('task1', 'task2', 'task3').
        task_config: Task metadata dict from tasks.py registry.
        collection: Current mutable in-memory document list.
        original_collection: Immutable reference used by reset().
        target_schema: JSON Schema target for current task.
        step_count: Steps taken in current episode.
        last_score: Score after previous step (for delta reward).
        stagnation_counter: Consecutive steps with no score improvement.
        last_action_status: Status string from last dispatched action.
        done: Whether current episode has terminated.
        protected_keys: Keys that cannot be deleted (schema 'required' fields).
    """

    def __init__(self, task_id: str = "task1") -> None:
        """
        Initialise environment for a given task. Loads static dataset from disk.

        Args:
            task_id: Which task to load. One of 'task1', 'task2', 'task3'.
        """
        self.task_id = task_id
        self.task_config = get_task(task_id)
        self.target_schema: Dict[str, Any] = self.task_config["target_schema"]
        self.protected_keys: List[str] = self.target_schema.get("required", [])

        with open(self.task_config["data_file"], "r") as f:
            self.original_collection: List[Dict] = json.load(f)

        self.collection: List[Dict] = copy.deepcopy(self.original_collection)
        self.step_count: int = 0
        self.last_score: float = 0.0
        self.stagnation_counter: int = 0
        self.last_action_status: str = "reset"
        self.done: bool = False

    def reset(self) -> SchemaObservation:
        """
        Reset environment to initial state. Reloads original collection from memory.

        Returns:
            Initial SchemaObservation before any actions are taken.
        """
        self.collection = copy.deepcopy(self.original_collection)
        self.step_count = 0
        self.last_score = calculate_score(self.collection, self.target_schema)
        self.stagnation_counter = 0
        self.last_action_status = "reset"
        self.done = False

        return build_observation(
            collection=self.collection,
            target_schema=self.target_schema,
            step_count=self.step_count,
            max_steps=self.task_config["max_steps"],
            current_score=self.last_score,
            last_action_status=self.last_action_status,
        )

    def step(self, action: SchemaAction) -> Tuple[SchemaObservation, float, bool, Dict]:
        """
        Apply one agent action and advance environment state.

        Args:
            action: SchemaAction with action_type and params.

        Returns:
            Tuple of (observation, reward_value, done, info).
            reward_value is a float: max(0, (S_t - S_{t-1})) * 10.
        """
        if self.done:
            obs = build_observation(
                collection=self.collection,
                target_schema=self.target_schema,
                step_count=self.step_count,
                max_steps=self.task_config["max_steps"],
                current_score=self.last_score,
                last_action_status="episode_already_done",
            )
            return obs, 0.0, True, {"info": "Episode already terminated"}

        self.step_count += 1
        info: Dict[str, Any] = {"step": self.step_count}

        if action.action_type == "terminate":
            self.done = True
            self.last_action_status = "success"
            obs = build_observation(
                collection=self.collection,
                target_schema=self.target_schema,
                step_count=self.step_count,
                max_steps=self.task_config["max_steps"],
                current_score=self.last_score,
                last_action_status=self.last_action_status,
            )
            info["termination_reason"] = "agent_terminated"
            return obs, 0.0, True, info

        pre_action_collection = copy.deepcopy(self.collection)

        status = dispatch_action(
            collection=self.collection,
            action_type=action.action_type,
            params=action.params,
            protected_keys=self.protected_keys,
        )
        self.last_action_status = status

        if status.startswith("error"):
            self.collection = pre_action_collection

        new_score = calculate_score(self.collection, self.target_schema)
        delta = max(0.0, new_score - self.last_score)
        reward_value = round(delta * 10.0, 4)

        if delta <= 0.0:
            self.stagnation_counter += 1
        else:
            self.stagnation_counter = 0

        info["previous_score"] = self.last_score
        info["new_score"] = new_score
        info["stagnation_counter"] = self.stagnation_counter

        self.last_score = new_score

        if new_score >= 1.0:
            self.done = True
            info["termination_reason"] = "task_complete"
        elif self.stagnation_counter >= STAGNATION_LIMIT:
            self.done = True
            info["termination_reason"] = "stagnation"
        elif self.step_count >= self.task_config["max_steps"]:
            self.done = True
            info["termination_reason"] = "max_steps_reached"

        obs = build_observation(
            collection=self.collection,
            target_schema=self.target_schema,
            step_count=self.step_count,
            max_steps=self.task_config["max_steps"],
            current_score=new_score,
            last_action_status=self.last_action_status,
        )
        return obs, reward_value, self.done, info

    def state(self) -> Dict[str, Any]:
        """
        Return full internal state for debugging and OpenEnv spec compliance.

        Returns:
            Serializable dict with complete environment state.
        """
        return {
            "task_id": self.task_id,
            "collection_size": len(self.collection),
            "target_schema": self.target_schema,
            "step_count": self.step_count,
            "last_score": self.last_score,
            "stagnation_counter": self.stagnation_counter,
            "done": self.done,
            "last_action_status": self.last_action_status,
        }
```

## 6. Action Space

### `server/actions.py`

```python
"""
actions.py
----------
All action implementations for Schema Surgeon.
Each action is:
- Global: applied to every document in the collection
- Idempotent: applying same action twice produces same result
- Deterministic: no randomness, no I/O

Actions:
    rename_and_merge(collection, source, target)
    cast_type(collection, key, target_type, default_value)
    flatten_field(collection, parent_key, child_key)
    delete_key(collection, key, protected_keys)
    dispatch_action(collection, action_type, params, protected_keys)

All handlers return a status string: "success" | "no_op" | "error: <message>"
"""

from typing import Any, Dict, List, Optional


def rename_and_merge(
    collection: List[Dict],
    source: str,
    target: str,
) -> str:
    """
    Globally rename 'source' key to 'target' across all documents.
    If both exist in a document, 'target' wins and 'source' is discarded.
    If only 'source' exists, it is moved to 'target'.

    Args:
        collection: In-place mutable list of document dicts.
        source: Key to rename or discard.
        target: Key to keep or create.

    Returns:
        "success" if at least one document was modified, else "no_op".
    """
    modified = 0
    for doc in collection:
        if source in doc:
            if target not in doc:
                doc[target] = doc.pop(source)
            else:
                del doc[source]
            modified += 1
    return "success" if modified > 0 else "no_op"


def cast_type(
    collection: List[Dict],
    key: str,
    target_type: str,
    default_value: Any = None,
) -> str:
    """
    Cast all values of 'key' to target_type across the collection.
    Supported types: "int", "float", "str".
    Failed casts are replaced with default_value (None → null in JSON).

    Args:
        collection: In-place mutable list of document dicts.
        key: Document key whose values should be cast.
        target_type: One of "int", "float", "str".
        default_value: Fallback when cast fails. Defaults to None.

    Returns:
        "success" if at least one value was changed.
        "no_op" if key not found in any document.
        "error: ..." if target_type is unsupported.
    """
    type_map = {"int": int, "float": float, "str": str}
    if target_type not in type_map:
        return f"error: unsupported target_type '{target_type}'. Use int, float, or str."

    caster = type_map[target_type]
    modified = 0

    for doc in collection:
        if key not in doc:
            continue
        original = doc[key]
        try:
            casted = caster(original)
            if casted != original or type(casted) != type(original):
                doc[key] = casted
                modified += 1
        except (ValueError, TypeError):
            doc[key] = default_value
            modified += 1

    return "success" if modified > 0 else "no_op"


def flatten_field(
    collection: List[Dict],
    parent_key: str,
    child_key: str,
) -> str:
    """
    Lift a nested field to root level of each document.
    Example: flatten_field("metadata", "version") transforms
        {"metadata": {"version": "v1"}} → {"version": "v1"}
    Parent key is removed if it becomes empty after extraction.
    If child_key already exists at root, it is overwritten.

    Args:
        collection: In-place mutable list of document dicts.
        parent_key: The nested parent key (e.g., "metadata").
        child_key: The child key inside parent to promote (e.g., "version").

    Returns:
        "success" if at least one document was modified, else "no_op".
    """
    modified = 0
    for doc in collection:
        if parent_key in doc and isinstance(doc[parent_key], dict):
            nested = doc[parent_key]
            if child_key in nested:
                doc[child_key] = nested.pop(child_key)
                if not nested:
                    del doc[parent_key]
                modified += 1
    return "success" if modified > 0 else "no_op"


def delete_key(
    collection: List[Dict],
    key: str,
    protected_keys: List[str],
) -> str:
    """
    Delete a key from all documents globally.
    Blocked if the key appears in protected_keys (schema 'required' fields).

    Args:
        collection: In-place mutable list of document dicts.
        key: Key to delete across all documents.
        protected_keys: Keys that must not be deleted.

    Returns:
        "success" if at least one document was modified.
        "no_op" if key not found in any document.
        "error: ..." if key is protected.
    """
    if key in protected_keys:
        return (
            f"error: '{key}' is protected (required by target schema) "
            "and cannot be deleted."
        )

    modified = 0
    for doc in collection:
        if key in doc:
            del doc[key]
            modified += 1
    return "success" if modified > 0 else "no_op"


def dispatch_action(
    collection: List[Dict],
    action_type: str,
    params: Dict[str, Any],
    protected_keys: List[str],
) -> str:
    """
    Route an action by name to its handler function.

    Args:
        collection: In-place mutable document list.
        action_type: One of rename_and_merge | cast_type | flatten_field | delete_key.
        params: Keyword arguments for the handler.
        protected_keys: Keys that cannot be deleted.

    Returns:
        Status string: "success" | "no_op" | "error: ..."
    """
    if action_type == "rename_and_merge":
        source = params.get("source")
        target = params.get("target")
        if not source or not target:
            return "error: rename_and_merge requires 'source' and 'target' params."
        return rename_and_merge(collection, source, target)

    elif action_type == "cast_type":
        key = params.get("key")
        target_type = params.get("target_type")
        default_value = params.get("default_value", None)
        if not key or not target_type:
            return "error: cast_type requires 'key' and 'target_type' params."
        return cast_type(collection, key, target_type, default_value)

    elif action_type == "flatten_field":
        parent_key = params.get("parent_key")
        child_key = params.get("child_key")
        if not parent_key or not child_key:
            return "error: flatten_field requires 'parent_key' and 'child_key' params."
        return flatten_field(collection, parent_key, child_key)

    elif action_type == "delete_key":
        key = params.get("key")
        if not key:
            return "error: delete_key requires 'key' param."
        return delete_key(collection, key, protected_keys)

    else:
        return f"error: unknown action_type '{action_type}'."
```

## 7. Observation Space

### `server/obs.py`

```python
"""
obs.py
------
Observation builder for Schema Surgeon.
Constructs SchemaObservation at each step from current environment state.

Design:
- sample_docs = first 10 docs (Golden Sample, deterministic, always same order)
- global_keys = presence ratio of each key across full 50-doc collection
- current_score is exposed to help agent track incremental progress
"""

from typing import Any, Dict, List

from server.models import SchemaObservation

GOLDEN_SAMPLE_SIZE: int = 10


def build_global_keys(collection: List[Dict]) -> Dict[str, float]:
    """
    Compute presence ratio of every key across all documents.
    Ratio = (docs containing key) / (total docs).

    Args:
        collection: Full document collection.

    Returns:
        Dict mapping key names to presence ratio (0.0-1.0), rounded to 4 decimals.
        Example: {"user_id": 0.7, "uid": 0.2, "u_id": 0.1}
    """
    if not collection:
        return {}

    key_counts: Dict[str, int] = {}
    total = len(collection)

    for doc in collection:
        for key in doc.keys():
            key_counts[key] = key_counts.get(key, 0) + 1

    return {k: round(v / total, 4) for k, v in key_counts.items()}


def build_observation(
    collection: List[Dict],
    target_schema: Dict[str, Any],
    step_count: int,
    max_steps: int,
    current_score: float,
    last_action_status: str,
) -> SchemaObservation:
    """
    Construct the full observation from current environment state.

    Args:
        collection: Current full document collection (50 docs).
        target_schema: JSON Schema target for this task.
        step_count: Current step number.
        max_steps: Hard cap for this task.
        current_score: Current fraction of docs passing schema (0.0-1.0).
        last_action_status: "success" | "no_op" | "error: ..." | "reset"

    Returns:
        SchemaObservation ready for serialization and return to agent.
    """
    sample_docs = collection[:GOLDEN_SAMPLE_SIZE]
    global_keys = build_global_keys(collection)

    return SchemaObservation(
        sample_docs=sample_docs,
        global_keys=global_keys,
        target_schema=target_schema,
        step_count=step_count,
        max_steps=max_steps,
        current_score=current_score,
        last_action_status=last_action_status,
    )
```

## 8. Reward Function

```
R_t = max(0, (S_t - S_{t-1})) * 10
```

- S = valid_docs / total_docs
- Reward only for net positive improvement
- Max cumulative reward = 10.0 (when S goes 0.0 → 1.0)
- Computed in `SchemaSurgeon_environment.py` after every step
- Score recomputed from scratch by `grader.calculate_score()` — no caching

## 10. Grader Implementation

### `server/grader.py`

```python
"""
grader.py
---------
Scoring logic for Schema Surgeon.
Uses jsonschema Draft 7 validation as the deterministic scoring engine.
Score S = (docs passing validation) / (total docs), range [0.0, 1.0].

Never catches broad exceptions — only jsonschema.ValidationError.
Never caches — always recomputes fresh from the collection state.
"""

import jsonschema
from jsonschema import ValidationError
from typing import Any, Dict, List


def calculate_score(
    collection: List[Dict[str, Any]],
    schema: Dict[str, Any],
) -> float:
    """
    Compute the fraction of documents that pass JSON Schema validation.

    Args:
        collection: Current list of document dicts (50 items).
        schema: Target JSON Schema (Draft 7 format).

    Returns:
        Float in [0.0, 1.0]. 1.0 means all documents are valid.
        Returns 0.0 if collection is empty.
    """
    if not collection:
        return 0.0

    valid_count = 0
    for doc in collection:
        try:
            jsonschema.validate(instance=doc, schema=schema)
            valid_count += 1
        except ValidationError:
            continue

    return round(valid_count / len(collection), 6)
```

## 13. Baseline Inference Script (`inference.py`)

```python
"""
inference.py
------------
Baseline inference script for Schema Surgeon.
Runs an LLM agent (via OpenAI-compatible client) against all 3 tasks sequentially.
Uses the SchemaSurgeonEnv client via async WebSocket context manager.

Environment variables (all required):
    API_BASE_URL   — LLM API endpoint base URL
    MODEL_NAME     — Model identifier string
    HF_TOKEN       — API key (used as OpenAI API key)
    ENV_BASE_URL   — Running environment server URL (default: http://localhost:7860)

Log format strictly follows [START], [STEP], [END] as required by hackathon evaluator.
Do NOT modify log field names, order, or format.

Usage:
    python inference.py
"""

import asyncio
import json
import os
from typing import Any, Dict, List, Optional

from openai import OpenAI

from client import SchemaSurgeonEnv
from server.models import SchemaAction

# ── Configuration ─────────────────────────────────────────────────────────────
API_BASE_URL: str = os.environ.get("API_BASE_URL", "https://api.openai.com/v1")
MODEL_NAME: str = os.environ.get("MODEL_NAME", "gpt-4o-mini")
API_KEY: str = os.environ.get("HF_TOKEN") or os.environ.get("OPENAI_API_KEY", "")
ENV_BASE_URL: str = os.environ.get("ENV_BASE_URL", "http://localhost:7860")

TEMPERATURE: float = 0.0
MAX_TOKENS: int = 512
MAX_STEPS: int = 30
SUCCESS_THRESHOLD: float = 0.95
TASKS: List[str] = ["task1", "task2", "task3"]

# ── Logging ────────────────────────────────────────────────────────────────────

def log_start(task: str, env: str, model: str) -> None:
    """Emit [START] log line as required by hackathon automated evaluator."""
    print(f'[START] task="{task}" env="{env}" model="{model}"', flush=True)


def log_step(step: int, action: Dict, reward: float, done: bool, error: Optional[str]) -> None:
    """Emit [STEP] log line as required by hackathon automated evaluator."""
    action_str = json.dumps(action)
    error_str = f' error="{error}"' if error else ""
    print(
        f'[STEP] step={step} action={action_str} reward={reward:.4f} done={done}{error_str}',
        flush=True,
    )


def log_end(success: bool, steps: int, score: float, rewards: List[float]) -> None:
    """Emit [END] log line as required by hackathon automated evaluator."""
    rewards_str = json.dumps([round(r, 4) for r in rewards])
    print(
        f'[END] success={str(success).lower()} steps={steps} score={score:.4f} rewards={rewards_str}',
        flush=True,
    )


# ── LLM Agent ─────────────────────────────────────────────────────────────────

SYSTEM_PROMPT = """You are a Database Migration Agent. You will receive:
1. A sample of 10 documents from a messy NoSQL collection.
2. A dict showing what keys exist globally and their prevalence ratios.
3. A target JSON Schema that ALL documents must satisfy.
4. Your current score (fraction of docs passing the schema).

Your goal: issue ONE action per turn to migrate the collection toward the target schema.

Available actions (respond ONLY with valid JSON, no markdown wrapping):
{"action_type": "rename_and_merge", "params": {"source": "<old_key>", "target": "<new_key>"}}
{"action_type": "cast_type", "params": {"key": "<key>", "target_type": "int|float|str", "default_value": <value>}}
{"action_type": "flatten_field", "params": {"parent_key": "<parent>", "child_key": "<child>"}}
{"action_type": "delete_key", "params": {"key": "<key>"}}
{"action_type": "terminate", "params": {}}

Think step by step:
1. What does the target schema require?
2. What is wrong in the current sample docs?
3. What single action makes the most progress?

Output ONLY the JSON action on the last line of your response."""


def get_agent_action(
    client: OpenAI,
    observation: Dict[str, Any],
    history: List[str],
) -> Dict[str, Any]:
    """
    Query LLM for next action given current observation.

    Args:
        client: Configured OpenAI client instance.
        observation: Current SchemaObservation as dict.
        history: List of recent step summaries for context window.

    Returns:
        Parsed action dict. Falls back to terminate on any parse failure.
    """
    history_str = "\n".join(history[-5:]) if history else "None"

    user_prompt = (
        f"Sample docs (first 10):\n{json.dumps(observation['sample_docs'], indent=2)}\n\n"
        f"Global key presence: {json.dumps(observation['global_keys'])}\n\n"
        f"Target schema: {json.dumps(observation['target_schema'])}\n\n"
        f"Current score: {observation['current_score']}\n"
        f"Step: {observation['step_count']} / {observation['max_steps']}\n"
        f"Last action status: {observation['last_action_status']}\n\n"
        f"Recent history:\n{history_str}\n\n"
        "Output your next action as JSON:"
    )

    try:
        completion = client.chat.completions.create(
            model=MODEL_NAME,
            messages=[
                {"role": "system", "content": SYSTEM_PROMPT},
                {"role": "user", "content": user_prompt},
            ],
            temperature=TEMPERATURE,
            max_tokens=MAX_TOKENS,
        )
        text = (completion.choices[0].message.content or "").strip()
        last_brace = text.rfind("{")
        if last_brace >= 0:
            return json.loads(text[last_brace:])
    except Exception as exc:
        print(f"[DEBUG] LLM parse error: {exc}", flush=True)

    return {"action_type": "terminate", "params": {}}


# ── Task Runner ────────────────────────────────────────────────────────────────

async def run_task(client: OpenAI, task_id: str) -> float:
    """
    Run the agent on a single task from reset to termination.

    Args:
        client: Configured OpenAI client.
        task_id: One of 'task1', 'task2', 'task3'.

    Returns:
        Final score for this task (0.0-1.0).
    """
    log_start(task=task_id, env="schema-surgeon", model=MODEL_NAME)

    async with SchemaSurgeonEnv(base_url=ENV_BASE_URL) as env_client:
        result = await env_client.reset()
        observation = result.observation.model_dump()

        history: List[str] = []
        rewards: List[float] = []
        steps_taken = 0
        final_score = 0.0
        success = False

        try:
            for step in range(1, MAX_STEPS + 1):
                action_dict = get_agent_action(client, observation, history)
                action = SchemaAction(**action_dict)

                result = await env_client.step(action)
                reward_val = result.reward if isinstance(result.reward, float) else 0.0
                done = result.done
                obs_dict = result.observation.model_dump()
                final_score = obs_dict["current_score"]
                error = None

                rewards.append(reward_val)
                steps_taken = step
                observation = obs_dict

                log_step(step=step, action=action_dict, reward=reward_val, done=done, error=error)
                history.append(
                    f"Step {step}: {action_dict['action_type']}({action_dict['params']}) "
                    f"-> reward={reward_val:.4f} score={final_score:.4f}"
                )

                if done:
                    break

            success = final_score >= SUCCESS_THRESHOLD

        finally:
            log_end(success=success, steps=steps_taken, score=final_score, rewards=rewards)

    return final_score


# ── Main ───────────────────────────────────────────────────────────────────────

async def main() -> None:
    """Entry point. Runs agent on all 3 tasks sequentially and prints summary."""
    if not API_KEY:
        raise EnvironmentError("Set HF_TOKEN or OPENAI_API_KEY environment variable.")

    llm_client = OpenAI(base_url=API_BASE_URL, api_key=API_KEY)
    all_scores: Dict[str, float] = {}

    for task_id in TASKS:
        print(f"\n{'='*50}", flush=True)
        print(f"Running {task_id}...", flush=True)
        score = await run_task(llm_client, task_id)
        all_scores[task_id] = score
        print(f"Final score for {task_id}: {score:.4f}", flush=True)

    print(f"\n{'='*50}", flush=True)
    print("All task scores:", json.dumps(all_scores, indent=2), flush=True)
    avg = sum(all_scores.values()) / len(all_scores)
    print(f"Average score: {avg:.4f}", flush=True)


if __name__ == "__main__":
    asyncio.run(main())
```
